Here's some of the new lingo you'll here.

Functional Interface: This is an interface with a single method. Also know as a Single Abstract Method type. There are many examples in the standard API, e.g. Runnable, ActionListener and Comparator.

Lambda expression: This is the new syntax for defining anonymous functions something which Java has needed for a long time e.g.
